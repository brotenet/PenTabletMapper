# PenTabletMapper

[![Download](https://img.shields.io/badge/Download-latest-blue)](https://github.com/brotenet/PenTabletMapper/releases)


Linux tool for mapping the stylus pen of a graphis tablet to a selected display.
- This application uses "xinput" for the mapping and it is only compatible with Linux OSes using XOrg or XLibre.
- This application requires Java v.25.0.2 (or later version) to be installed on the system.
